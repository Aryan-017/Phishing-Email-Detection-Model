# Phishing Email Detection Model

A Machine Learning based cybersecurity project that detects phishing emails using Natural Language Processing and Scikit-learn.

## Features
- Detects phishing and safe emails
- TF-IDF feature extraction
- Naive Bayes classification
- GUI-based email analyzer
- Accuracy score and confusion matrix
- Real-time email prediction

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Tkinter
- NLP (Natural Language Processing)

## Machine Learning Concepts Used
- Text preprocessing
- TF-IDF Vectorization
- Train-Test Split
- Naive Bayes Algorithm
- Confusion Matrix Evaluation

## Educational Purpose
This project was developed to understand:
- Email phishing attacks
- Machine learning in cybersecurity
- NLP-based threat detection
- Basic AI model training
